Removed sensitive keys/tokens from this shared copy.

Before running the project, fill these placeholders with your own values:
1. gradle.properties -> MAPBOX_DOWNLOADS_TOKEN
2. app/src/main/res/values/strings.xml -> mapbox_access_token
3. app/src/main/AndroidManifest.xml -> com.google.android.geo.API_KEY
4. app/google-services.json -> current_key (or replace the whole file with your Firebase project file)
